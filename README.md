# Nexsus — The substrate

**Private, persistent memory for your AI assistant.**

Nexsus is a private workspace that gives your AI assistant a memory that lasts. Connect Claude, ChatGPT, or Gemini over MCP and your assistant gains a persistent, structured, semantically searchable memory — in an isolated workspace that's yours alone.

**This is a remote MCP server** — nothing to install or run. This repository is the public home for the listing; the service itself is hosted.

## Connect

- **Endpoint:** `https://app.nexsus.com.au/mcp` (Streamable HTTP)
- **Auth:** OAuth 2.1 — sign in with your Nexsus workspace (no API keys pasted into chat)
- **Works with:** Claude, ChatGPT, Gemini — any MCP-capable client
- **Get a workspace:** https://www.nexsus.com.au — free plan, no card required

## What it does

- Persistent memory across conversations — nothing is lost between sessions
- Semantic search over everything you've stored
- Structured records with full version history — every edit is preserved
- Private, isolated per-tenant workspace
- Zero-code setup — add the connector URL and sign in
