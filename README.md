# Nexsus — The substrate

> Nexsus is a private workspace that gives your AI assistant a memory that lasts.

Connect your AI assistant to Nexsus and it gains a persistent, structured memory inside a private, isolated workspace that's yours alone. Works with Claude, ChatGPT and Gemini over the open MCP standard.

## What you get

- Persistent, structured memory your AI assistant can search semantically
- A private, isolated workspace that is yours alone
- Works with Claude, ChatGPT and Gemini over the open MCP standard
- A full versioned history of every change
